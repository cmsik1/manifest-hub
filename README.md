# MANIFEST-HUB

이 저장소는 AI 에이전트의 페르소나와 워크플로우를 체계적으로 관리하기 위한 중앙 Orchestration Layer입니다. 사용자의 의도에 따라 에이전트가 일관된 결과물을 도출할 수 있도록 특정 도메인에 최적화된 가이드라인과 제약 사항(Constraints)을 정의합니다.

## 1. 개요 (Overview)

MANIFEST-HUB는 AI/AX 업무 효율화를 위해 에이전트의 System Role과 작동 지침을 구조화합니다. 현재는 반도체 산업 기술 학습 및 주식 시장 분석 데이터를 블로그 포스팅으로 전환하는 워크플로우를 핵심적으로 다루고 있습니다.[cite: 1]
